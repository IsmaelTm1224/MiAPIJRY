01. SISTEMA DE APIs

¿Qué es una API?

Una API es un sistema de comunicación entre software. Permite enviar solicitudes, recibir respuestas y conectar servicios sin intervención humana.

REST

Modelo basado en HTTP (GET, POST, PUT, DELETE).

GraphQL

Consulta exacta de datos. Más eficiente que REST.

gRPC

Comunicación ultrarrápida con Protocol Buffers.
